# Mis proyectos y análisis

Este directorio servirá como espacio común donde recopilar todo lo relacionado con tus proyectos y análisis. Podemos crear aquí subcarpetas por tema o fecha, y mantener un índice actualizado.

Puedes comenzar creando carpetas nombradas con el proyecto (por ejemplo `proyecto-x`) y agregando archivos `README.md` dentro para describir el objetivo, las decisiones principales y los elementos pendientes.

Cuando quieras trabajar en alguno de ellos, dime cuál quieres desplegar y lo levantamos en el entorno local.
